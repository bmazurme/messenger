
# Project Messenger

### About
* JS/TS web messenger project, WebSocket

Implemented features:
- [X] User registration
- [X] User authorization
- [X] Edit user profile
- [X] Uploading avatar
- [X] Create chat
- [X] User search
- [X] Adding a user to a chat
- [X] Form Validation

- `npm run start` — launch,
- `npm run build` — build.
